# Infrastructure

Responsável pelas implementações técnicas do sistema.

## Componentes

### Repositories
Implementações concretas dos contratos definidos no domínio.

### Persistence
Armazenamento em memória utilizado durante a execução.

### DependencyInjection
Registro de serviços e dependências.

## Observação

Atualmente o projeto utiliza persistência em memória através do
MemoryContext. A troca para banco de dados deve ocorrer apenas
nesta camada.
